This is an experiment. I'm trying to re-record [this talk](https://www.youtube.com/watch?v=xEu3t-KJVVg) using [Remotion](https://www.remotion.dev/).

## Commands

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Server render demo**

```console
npm run server
```
